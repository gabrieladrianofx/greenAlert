# GreenAlert

## Descrição

Projeto para realização da disciplina 'Projeto de Extensão II'

## Instalação

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```

2. Entre na pasta do projeto:

   ```bash
    cd caminho-onde-esta-localizado-o-nome-da-pasta
   ```

## Execução

Para iniciar o projeto, execute o comando abaixo:

  ```bash
  php -S localhost:8888 -d auto_prepend_file=server.php -t public
  ```

Depois disso, abra seu navegador e acesse:

  ```bash
  localhost:8888
  ```
